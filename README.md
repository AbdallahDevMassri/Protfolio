To run this website you need to follow this steps
1-using the trminal <npm install>
2-install the framework of nextjs tailwindcss using this commands <npm install -D tailwindcss postcss autoprefixer>
<npx tailwindcss init -p>
3-and configure your template in the tailwind.config.js
content: [
// using `src` directory:
"./src/\*_/_.{js,ts,jsx,tsx,mdx}",

],
4-adding the talwind css inside src/styles/global.css
@tailwind base;
@tailwind components;
@tailwind utilities;

5- start your build process
run this command <npm run dev>
